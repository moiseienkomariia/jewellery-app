Projekt dzięki któremu nauczysz się podstaw Reacta oraz Reduxa.

W ramach tego projektu będziemy próbowali zaimplementować tę aplikację:
https://www.figma.com/design/fv0g684EubdzP4ZQ1d7UDX/Jewellery-App-%F0%9F%92%8E--1-?node-id=0-1&p=f&t=tbK5JgCaz8nbItYA-0

Do kazdego zadania twórz nowego brancha w formacie: feature/task-X-title, po ukończeniu zadania utwórz pull request do brancha main.

Setup:

- npm install
- npm run dev
